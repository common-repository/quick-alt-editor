Edit your alt very quick without reloading

== Description ==

The plugin add a field to images in Media Library. You can edit ALT text without reloading, with AJAX.
Enter text and save it with blur input field. So simple!

Compatible with `qTranslate-X`, `qTranslate-XT`.

&copy; [Arsmoon - Top rated web development company, based in Ukraine](https://arsmoon.com/ "Arsmoon")

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/quick-alt-editor` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Media Library, choose 'List view', enter text in `Alternative Text` field
